# GitHub Pages 포트폴리오 v3

자격증·교육·경력 표에 있는 구체적인 날짜, 기관, 교육시간을 반영한 개정본입니다.

## 추가 반영

- 자격증 취득일과 발급기관
- 교육기관, 기간, 시간, 교육내용
- 연구·경력 수행기관과 기간
- 자격증 10개, 교육·활동 5개, 확인된 교육시간 88시간 요약
- 자격번호는 공개 페이지에서 제외

## 개인정보 처리

자격번호는 고유 식별 정보이므로 공개 GitHub Pages에는 넣지 않았습니다.
증빙 PDF 역시 공개 저장소보다는 입사지원 시 개별 제출하는 편이 안전합니다.
