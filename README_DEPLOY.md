# 개인 이력서 포트폴리오 v8

학력·어학 구간의 빈 공간을 줄이고, 데스크톱 호환성과 현재 학습 기록 배치를 개선한 버전입니다.

## 변경 사항

### 학력·핵심 자격·어학
한 줄에 다음 세 영역을 배치했습니다.

1. EDUCATION
2. KEY CERTIFICATIONS
3. LANGUAGE

핵심 자격은 지원 직무와 연관성이 높은 다음 4개만 우선 표시합니다.

- 빅데이터분석기사
- 도시계획기사
- Autodesk Certified User: AutoCAD®
- 컴퓨터활용능력 1급

전체 자격증 10개는 기존 자격·교육 구간에 그대로 유지됩니다.

### 현재 학습 기록
9개 카드를 무작위로 나열하지 않고 다음 3개 그룹으로 재구성했습니다.

- 자격시험 준비
- 데이터·통계
- AI·공간정보

각 그룹 안에서 관련 학습 저장소를 세로 목록으로 확인할 수 있습니다.

### 데스크톱 브라우저
기존에는 781px~1050px 사이에서 데스크톱 메뉴가 사라지고 모바일 메뉴 버튼도 나타나지 않는 브레이크포인트 문제가 있었습니다.

v8에서는 다음과 같이 수정했습니다.

- 1120px 초과: 일반 데스크톱 메뉴
- 1120px 이하: 햄버거 메뉴
- 780px 이하: 모바일 1열 구성
- 데스크톱에서 본문과 프로필 영역을 명시적으로 표시
