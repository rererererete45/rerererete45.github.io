# 이력서형 GitHub Pages 포트폴리오 v4 — Light

Stitch의 밝고 전문적인 분위기를 참고해 흰색·연한 회색·파란색 중심으로 재설계했습니다.

## 주요 변경

- 어두운 배경 제거
- 흰색·연한 회색 기반 이력서형 디자인
- 파란색을 주 강조색으로 사용
- 프로필 사진 카드 추가
- 경력 타임라인과 자격증 표를 이력서처럼 정리
- 프로젝트 카드와 모바일 하단 메뉴 유지

## 프로필 사진 넣기

1. 사용할 사진을 준비합니다.
2. 파일명을 정확히 `profile.jpg`로 변경합니다.
3. `index.html`, `style.css`, `script.js`와 같은 위치에 업로드합니다.

```text
rerererete45.github.io/
├─ index.html
├─ style.css
├─ script.js
├─ profile-placeholder.svg
└─ profile.jpg
```

`profile.jpg`가 없으면 기본 자리표시 이미지가 표시됩니다.

## 권장 사진

- 세로형 증명사진 또는 상반신 사진
- 권장 비율: 4:5
- 권장 해상도: 800×1000px 이상
- 단색 또는 복잡하지 않은 배경
- 얼굴이 중앙보다 약간 위에 오도록 배치

## 업로드

기존 저장소에서 다음 파일을 교체합니다.

```text
index.html
style.css
script.js
```

그리고 새로 다음 파일을 추가합니다.

```text
profile-placeholder.svg
profile.jpg
```

자격번호와 증빙 PDF는 공개 저장소에 올리지 않는 것을 권장합니다.
