# GitHub Pages 포트폴리오 파일

이 폴더에는 GitHub Pages에 바로 업로드할 수 있는 파일이 들어 있습니다.

## 파일 구성

- `index.html`: 사이트 내용과 구조
- `style.css`: 색상, 배치, 반응형 디자인
- `script.js`: 모바일 메뉴 동작

## 업로드 위치

저장소 이름:

```text
rerererete45.github.io
```

세 파일을 저장소 최상위(root)에 업로드합니다.

```text
rerererete45.github.io/
├─ index.html
├─ style.css
└─ script.js
```

## GitHub Pages 설정

1. 저장소의 `Settings` 선택
2. 왼쪽 메뉴에서 `Pages` 선택
3. `Build and deployment`의 `Source`를 `Deploy from a branch`로 설정
4. Branch를 `main`, 폴더를 `/(root)`로 설정
5. `Save` 선택
6. 잠시 후 `https://rerererete45.github.io` 접속

## 수정할 곳

- 소개 문구: `index.html`의 `hero`, `about` 영역
- 기술: `index.html`의 `skills` 영역
- 프로젝트: `index.html`의 `projects` 영역
- 자격증과 학습 기록: `index.html`의 `learning` 영역
- 색상: `style.css` 맨 위 `:root` 변수
